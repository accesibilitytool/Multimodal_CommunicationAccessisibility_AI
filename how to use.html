Overview

Vita AI helps Blind, Mute, and Deaf users using Image Description, Text Reading, Live Camera Describing, Speaking Messages, and Speech-to-Text.

For Blind Users

Follow these steps to use Vita AI comfortably.

Image Recognition

  1. Tap Image Recognition.
  2. Upload any image you want described.
  3. Press Analyze Image. Vita AI will speak what is seen.

Text Recognition (Reading Text from Images)

  1. Tap Text Recognition.
  2. Upload an image containing text.
  3. Press Extract Text to hear the text aloud.

Live Vision (Real-Time Describing)

  1. Tap Live Vision Detection.
  2. Allow camera permission.
  3. Press Start Describing to hear surroundings.
  4. Press Stop Describing to end.

Real-Time Location

  1. Tap Get Real-Time Location.
  2. Allow location permission.
  3. Vita AI will speak your current address.

For Mute Users

Use Vita AI to speak messages for you.

  1. Select your preferred language.
  2. Type your message in the text box.
  3. Press Speak Message to make Vita AI speak.
  4. Use Quick Messages for common phrases.

For Deaf Users

Convert speech into written text in real-time.

  1. Select language.
  2. Press Start Speech-to-Text.
  3. Allow microphone permission.
  4. Spoken words will appear as text.
  5. Press Stop Speech-to-Text to finish.

Permissions Needed

  • Camera — For photo & live vision features.
  • Microphone — For speech-to-text.
  • Location — For real-time address detection.

Troubleshooting

  • No audio? Increase volume and allow sound permission.
  • Camera not working? Allow camera permission.
  • Speech-to-text not working? Allow microphone access.
  • Location not detected? Turn on GPS/location services.
